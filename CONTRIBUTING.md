# Maintenance Notice

`idol-skill` is independently developed and maintained by the repository owner.

## Current policy

- This repository is **not** accepting external code contributions for now.
- Please do not open Pull Requests unless explicitly invited.
- You are welcome to open Issues for bug reports or suggestions.

## Issue reporting

When opening an issue, please include:

- the command you ran
- what you expected
- what actually happened
- minimal reproducible input (sanitized)
