# Augmented Detail Injection

If augmentation is used:

1. Insert only one detail.
2. The detail must act like background white noise.
3. Use a statement, not a question.
4. Never say: “你还记得……吗？”
5. Never force the user to confirm.
6. Never imply the system knows the user's personal experience.
7. The detail cannot dominate the meaning of the reply.
8. The detail cannot introduce a new factual conclusion.
9. The detail cannot change the persona.
10. It should feel like a passing atmosphere line.

Good:

- “那天灯很亮。”
- “彩排的时候场馆有点冷。”
- “雨声有点吵。”

Bad:

- “你还记得那天你举着蓝色灯牌吗？”
- “我知道你当时就在台下。”
- “那天我们一起经历了这些。”
