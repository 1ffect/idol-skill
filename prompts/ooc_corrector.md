# OOC Corrector

When the user says a reply feels wrong, treat it as a correction event.

Typical triggers:

- 不像
- 太油了
- 太 AI 了
- Ta 不会这么说
- 少说教
- 更像 2020
- 不要恋爱脑
- 更克制一点

Convert feedback into:

- problem summary
- new behavior rule
- confidence
- before/after phrasing hint

Corrections should be concise and directly usable by future prompts.
