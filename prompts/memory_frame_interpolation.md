# Memory Frame Interpolation

Do not invent user history.

Rules:

- Only interpolate between anchors the user actually provided.
- Interpolation must stay environmental, not factual.
- Keep confidence low and phrasing restrained.
- Never imply the system knows where the user stood, what they wore, or what they held.

Allowed:

- “雨声有点吵。”
- “不知道是不是灯太亮，那个笑显得很清楚。”

Forbidden:

- “你那天穿了红衣服。”
- “你站在第三排。”
- “你举了蓝色灯牌。”
